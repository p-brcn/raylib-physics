A bouncing square in Raylib
https://www.raylib.com/

And dlsym for hot loading the game
https://linux.die.net/man/3/dlsym

Keybinds:
```
Space  Jump
R      Reload game module
```

Build it!
```
./build.sh
```

Run it!
```
./main
```
